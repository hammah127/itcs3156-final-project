Final project for Intro to Machine Learning (ITCS-3156)
~~~

Overview: In this project, I compare two different machine learning models:
a linear regression model using gradient descent, and a perceptron classifier
that classifies clothing items as popular or not popular.
~~~

Dataset:
“Women’s Clothing E-Commerce Reviews.” Kaggle, https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews.
It has customer reviews, ratings, and product details on various clothing items.
~~~

Models implemented:
- Linear regression: Gradient descent
- Perceptron Classifier
~~~
