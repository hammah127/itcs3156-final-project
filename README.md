Overview: In this project, I compare two different machine learning models: a linear regression model using gradient descent, and a perceptron classifier that classifies clothing items as popular or not popular.
~~~

Dataset:
“Women’s Clothing E-Commerce Reviews.” Kaggle, https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews.
It has customer reviews, ratings, and product details on various clothing items.
~~~

Features used:
- age
- positive feedback count
- department name
- class name
target variable: popular-> 1 if rating ≥ 4, and if not 0
~~~

Models implemented:
- Linear regression: Gradient descent
- Perceptron Classifier
~~~
